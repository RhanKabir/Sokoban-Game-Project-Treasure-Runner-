# Overview
The player navigates through a graph of interconnected rooms, pushing obstacles, collecting treasure, and traversing portals to progress. Game logic is handled entirely in C, while the frontend is built in Python using curses for real-time terminal rendering. The two layers are bridged through a ctypes wrapper.

# Features
Multi-room graph structure with player navigation
Obstacle pushing mechanics
Treasure collection
Portal traversal between rooms
Real-time terminal rendering

# How to run
 make
 python3 main.py
